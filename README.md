# Análisis de Vehículos Usados - tripautos

## Descripción
Esta aplicación web permite explorar y analizar un conjunto de datos de vehículos usados mediante visualizaciones interactivas.

## Funcionalidades
- Visualización de histogramas
- Gráficos de dispersión interactivos
- Filtros por marca y modelo

## Cómo usar
1. Ejecuta `streamlit run app.py`
2. Abre tu navegador en localhost:8501
